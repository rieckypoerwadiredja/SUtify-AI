## Teknologi yang Digunakan

- Python 3.7.6
- NLTK Library

## Cara Menjalankan Proyek

- 1. Install Python 3.7.6.
- 2. Install NLTK library dengan menggunakan pip command pip install nltk.
- 3. Jalankan file main.py untuk memulai proyek.
